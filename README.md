# 咔咔出行（出行可视化）

## 项目简介

解决出行问题，用于出行行程记录，路线规划，数据可视化分析的移动端webapp

## 技术栈

- **前端**：移动端，vue全家桶，Mand组件库，Scss

- **后端**：Node，Express框架
- **数据库**：Mysql

## 项目运行

``` bash
# 克隆项目
git clone git@github.com:wwenj/tripRecord.git

# 下载依赖
npm install

# 运行项目 打开端口localhost:8081
npm run server

# 项目打包
npm run build

```
